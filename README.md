## Getting started

1. Clone this repository

   ```bash
   git clone https://github.com/MoustafaElhadary/intros.git
   ```

2. Install dependencies

      ```bash
      yarn install
      ```
3. Create a **.env** file for the firebase config

    ```env
    API_KEY=XXXX
    AUTH_DOMAIN=XXXX
    PROJECT_ID=XXXX
    STORAGE_BUCKET=XXXX
    MESSAGING_SENDER_ID=XXXX
    APP_ID=XXXX
    ```
4. run 

      ```bash
      yarn start
      ```
